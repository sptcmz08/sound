# START HERE — เริ่มใช้งานตรงนี้

## ระบบนี้เขียนด้วยอะไร

- PHP 8.2+
- Laravel
- MySQL หรือ MariaDB
- Laravel Blade
- Bootstrap 5
- JavaScript หรือ Alpine.js

เป็น Web Application ภาษาไทย รองรับคอมพิวเตอร์ แท็บเล็ต และมือถือ และติดตั้งได้บน Plesk, DirectAdmin หรือ Ubuntu VPS

## วิธีเริ่ม

1. แตกไฟล์ ZIP
2. วางไฟล์ทั้งหมดไว้ใน Root ของโฟลเดอร์ที่จะสร้างระบบ
3. เปิดโฟลเดอร์ด้วย Antigravity, Cline, Codex หรือ AI Coding Agent
4. เปิด `RUN_PROMPT.md`
5. Copy เนื้อหาทั้งหมด
6. Paste ลงในช่องสั่งงาน AI
7. ให้ AI อ่าน `SKILL.md` และ `STOCK_SYSTEM_SPEC.md`
8. AI จะตรวจว่ามี Laravel Project หรือยัง และเริ่มสร้างระบบ

## ไฟล์สำคัญ

- `RUN_PROMPT.md` — Prompt ที่ใช้สั่ง AI ลงมือทำงาน
- `SKILL.md` — กฎควบคุม AI
- `STOCK_SYSTEM_SPEC.md` — ขอบเขตและ Requirement
- `MASTER_PROMPT.md` — Prompt ฉบับละเอียดเดิม
- `README_AI_USAGE.md` — วิธีใช้งานและ Prompt สำหรับแก้ Bug/ทำงานต่อ

## คำสั่งที่ต้องใช้

ให้ Copy ทั้งไฟล์ `RUN_PROMPT.md` ไปสั่ง AI ไม่ใช่แค่แนบไฟล์อย่างเดียว
